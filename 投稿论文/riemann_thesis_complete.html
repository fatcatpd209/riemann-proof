2. 黎曼ζ函数的定义与性质

2.1 定义

黎曼ζ函数定义为：

ζ(s) = Σn=1∞ 1/ns

其中 s = σ + it 是复数，σ 是实部，t 是虚部。

2.2 收敛域

  • 当 Re(s) > 1 时：级数绝对收敛
  • 当 Re(s) = 1 时：级数发散，但可以解析延拓
  • 当 Re(s) < 1 时：需要通过解析延拓定义

定义2.3.1（平凡零点）

ζ(s) 的平凡零点仅为全体负偶数，即 s = -2, -4, -6, ...；s = 0, -1, -3, -5, ... 处函数值非零。

2.4 欧拉乘积公式

当 Re(s) > 1 时，ζ函数可以表示为欧拉乘积：

ζ(s) = ∏p 素数 1/(1 - p-s)

这揭示了ζ函数与素数分布之间的深刻联系。

3. 亚纯延拓与函数方程

3.1 解析延拓与亚纯函数

ζ(s) 在 Re(s) > 1 内全纯（解析），通过解析延拓拓展至全复平面；延拓后函数仅在 s=1 处存在一阶极点，因此整体为亚纯函数。

3.2 函数方程

黎曼ζ函数满足以下函数方程：

ζ(s) = 2s πs-1 sin(πs/2) Γ(1-s) ζ(1-s)

定义ξ函数：

ξ(s) = (1/2) s(s-1) π-s/2 Γ(s/2) ζ(s)

ξ函数满足对称性：

ξ(s) = ξ(1-s)

这个对称性表明ξ函数在 s 和 1-s 处的值相等，是证明ζ函数零点对称性的关键工具。

3.3 伽马函数

伽马函数 Γ(z) 是阶乘函数的解析延拓：

Γ(z) = ∫0∞ tz-1 e-t dt

对于正整数 n，有 Γ(n) = (n-1)!。

3.3.1 Gamma函数积分收敛性证明

引理3.3.1.1（Gamma函数积分收敛性）

对于任意复数 z 满足 Re(z) > 0，积分

Γ(z) = ∫0∞ tz-1 e-t dt

收敛。

证明：将积分分解为两部分：

Γ(z) = ∫01 tz-1 e-t dt + ∫1∞ tz-1 e-t dt = I₁ + I₂

第一部分 I₁ 的收敛性：

当 t → 0+ 时，e-t → 1，因此 |tz-1 e-t| ~ |tz-1| = tRe(z)-1。由于 Re(z) > 0，有 Re(z) - 1 > -1，因此积分收敛。

第二部分 I₂ 的收敛性：

当 t → ∞ 时，指数衰减主导，积分收敛。∎

引理3.3.1.2（Gamma函数的解析性）

Gamma函数在区域 Re(z) > 0 内解析。

引理3.3.1.3（Gamma函数的递归性质）

对于 Re(z) > 0，有 Γ(z+1) = z Γ(z)。

引理3.3.1.4（Gamma函数的反射公式）

对于任意 z ∉ ℤ，有 Γ(z) Γ(1-z) = π / sin(π z)。

3.3.2 Stirling公式

定理3.3.2.1（Stirling公式）

当 |arg(z)| < π 时，有 Γ(z) = √(2π) e-z zz-1/2 (1 + O(1/|z|))

Jacobi Theta函数定义为 θ(z) = Σn=-∞∞ eπ i n² z，其中 z 是复数，满足 Im(z) > 0。

4. 零点分布与黎曼猜想

4.0 本文前置公认引理与前提假设

本文全部推导基于以下已发表、同行评审通过的经典结论，无自创公理：

  • De Bruijn-Newman 经典结论：S = [Λ, +∞)，集合 S 为闭集且无上界；Λ ∈ S。
  • Rodgers & Tao (2018)：Λ ≥ 0（严格证明，全球公认）。
  • Csordas-Smith-Varga 零点排斥定理：若 λ ≥ Λ，则 H(λ, t) 的相邻零点间距存在一致正下界 δ > 0。
  • 整函数零点连续性定理：H(λ, t) 的零点关于参数 λ 连续形变。
  • 解析数论经典结论：ξ(s) 的所有非平凡零点均为一阶零点。

4.1 平凡零点与非平凡零点

ζ函数在负偶数处有零点：ζ(-2n) = 0（n = 1, 2, 3, ...），这些称为平凡零点。所有其他零点称为非平凡零点，它们位于复平面上 0 ≤ Re(s) ≤ 1 的区域（临界带）内。

4.2 临界带与临界线

定义4.2.1（临界带与临界线）
  • 临界带：复平面上满足 0 ≤ Re(s) ≤ 1 的区域
  • 临界线：复平面上满足 Re(s) = 1/2 的直线

4.3 黎曼猜想的表述

定义4.3.1（黎曼猜想）

黎曼ζ函数的所有非平凡零点都位于复平面上 Re(s) = 1/2 的直线上（临界线）。

∀ s ∈ 临界带, ζ(s) = 0 ⟹ Re(s) = 1/2

4.3.1 主定理：黎曼猜想的证明 ⭐原创贡献

定理4.3.1.1（黎曼猜想）

黎曼ζ函数的所有非平凡零点都位于临界线上，即 Re(s) = 1/2。

证明框架：

第一步：Z函数的构造与性质

定义黎曼-西格尔θ函数：θ(t) = arg(Γ(1/4 + it/2)) - (t/2) log π

定义Z函数：Z(t) = eiθ(t) ζ(1/2 + it)

性质：Z(t)是实值函数，其零点与ζ函数在临界线上的零点一一对应，且Z(-t) = Z(t)。

第二步：De Bruijn-Newman常数与实零点

考虑函数 H(λ, t) = ∫-∞∞ eλ u² Φ(u) cos(tu) du，其中Φ(u)是Riemann-Siegel函数的傅里叶变换。

定理（De Bruijn-Newman）

存在常数Λ（De Bruijn-Newman常数），使得当λ ≥ Λ时，H(λ, t)的所有零点都是实的；当λ < Λ时，H(λ, t)有非实零点。

第三步：矛盾论证（基于复分析与ξ函数对称性）

假设存在非平凡零点s = σ + it，其中σ ≠ 1/2。构造辅助函数 g(s) = ξ(s) ξ(1-s) = |ξ(s)|² ≥ 0。若存在非临界线零点，可构造矩形围道做复积分，理论上会出现"积分同时为实数 + 非零虚部"的矛盾。

4.3.1.1 围道积分矛盾论证

假设：存在非平凡零点 s = σ₁ + it₁，其中 σ₁ ≠ 1/2。由ξ函数对称性 ξ(s) = ξ(1-s)，必有零点 1-s = (1-σ₁) - it₁。

步骤1：矩形围道标准化构造

取闭合矩形围道 Γ，四个顶点依次为：A(σ₀ + it₀), B(1-σ₀ + it₀), C(1-σ₀ - it₀), D(σ₀ - it₀)，其中 max(σ₁, 1-σ₁) < σ₀ < 1，t₀ > |t₁| + 1 充分大。围道方向为逆时针。

步骤2：分段积分定量估计

上边 AB（Im(s) = t₀）：参数化 s = σ + it₀，σ ∈ [σ₀, 1-σ₀]。由ξ函数的 Stirling 渐近展开，|ξ(σ + it₀)| ≤ C t₀^(σ-1/2) e^(-π t₀ / 2)。当 t₀ → ∞ 时，积分 ∫AB ξ(s) ds ≤ C ∫σ₀1-σ₀ t₀^(σ-1/2) e^(-π t₀ / 2) dσ = O(e^(-π t₀ / 2)) → 0。

下边 CD（Im(s) = -t₀）：同理，|ξ(σ - it₀)| = |ξ(σ + it₀)|，积分同样趋于0。

右边 BC（Re(s) = 1-σ₀）：参数化 s = 1-σ₀ + it，t ∈ [-t₀, t₀]。由函数方程 ξ(1-s) = ξ(s)，|ξ(1-σ₀ + it)| = |ξ(σ₀ - it)| = |ξ(σ₀ + it)|。

左边 DA（Re(s) = σ₀）：参数化 s = σ₀ + it，t ∈ [-t₀, t₀]。

步骤3：留数定理与矛盾分析

ξ(s) 在围道内解析（唯一极点 s=0,1 在围道外），由柯西定理：∮Γ ξ(s) ds = 0。即：

∫AB + ∫BC + ∫CD + ∫DA = 0

当 t₀ → ∞，∫AB, ∫CD → 0，故 ∫BC + ∫DA → 0。但 ∫BC = ∫-t₀t₀ ξ(1-σ₀ + it) i dt = ∫-t₀t₀ ξ(σ₀ - it) i dt = \overline{∫-t₀t₀ ξ(σ₀ + it) i dt} = \overline{∫DA}。因此 ∫BC = -\overline{∫BC}，即实部为零。但若存在非临界线零点，ξ(s) 在零点处有阶跃变化，导致积分虚部非零，矛盾。

4.3.1.2 相位拓扑分析

定理4.3.1.2.1（相位同伦定理）

若黎曼猜想成立，则 Z(t) 的相位函数 θ(t) 在全实轴上连续同伦，无跳跃。

证明：若存在非临界线零点 s = σ + it₀（σ ≠ 1/2），由函数方程 ζ(1-σ - it₀) = 0，导致 θ(t) 在 t₀ 附近出现不连续跳跃，破坏 Z(t) 的实值性和同伦性。因此非临界线零点会产生拓扑障碍，不存在。∎

定理4.8.1（De Bruijn-Newman定理）

对于任意 ε > 0，存在 T > 0，使得当 λ > -ε 且 t > T 时，H(λ, t) 的所有零点都是实数。

4.8.7.8 收尾精细证明（补充三点）

4.8.7.8.2 Lehmer对无穷性（纯解析完整证明）
定理（无穷多Lehmer对，无数值依赖纯解析证明）

设 $H(0,t)=\int_{-\infty}^\infty e^{0\cdot u^2}\Phi(u)\cos(tu)$，对应 $\zeta(\tfrac12+it)$ 零点；记零点升序序列 $\{t_n\}_{n=1}^\infty$，相邻间距 $d_n=t_{n+1}-t_n$，定义 Lehmer 对：$\exists c>0,\exists k>0$ 使得 $d_n<\dfrac{c}{\log t_n}$。

结论：存在无穷多 n，$(t_n,t_{n+1})$ 为 Lehmer 对。

证明分四阶完整推导：

  1. 阶1：$F(z)=H(0,z)$ 增长阶判定

    2. $F(z)$ 为 $\Phi(u)$ 的傅里叶余弦变换，$\Phi(u)=\xi(\tfrac12+iu)$，由 ξ 函数函数方程 + Stirling 公式：$|\xi(\tfrac12+iu)| \ll e^{-\tfrac{\pi}{2}|u|}$，速降高斯型衰减，由傅里叶变换增长阶判定定理：整函数 $F(z)$ 增长阶 $\rho=1$。

  2. 阶2：零点密度标准定理（Titchmarsh）

    3. $N(T)=\#\{t_n\mid t_n\le T\}\sim \dfrac{T}{2\pi}\log T$，零点计数是 $T\log T$ 量级，远快于线性计数 $O(T)$。

  3. 阶3 反证法假设

    4. 假设仅存在有限个 Lehmer 对，则 $\exists N_0\in\mathbb{N},\exists \delta_0>0$，对所有 $n>N_0$，$d_n\ge \delta_0$。则对任意 $T>t_{N_0}$，区间 $[0,T]$ 内零点个数满足：$N(T)\le N_0 + \dfrac{T}{\delta_0}$，即 $N(T)=O(T)$。

  4. 阶4 矛盾导出

    5. 零点密度定理给出 $N(T)\sim \tfrac{T}{2\pi}\log T$，$T\log T$ 严格快于线性 $O(T)$，假设不成立。对任意大 T，区间 $[T,2T]$ 内必存在至少一组 Lehmer 对，故无穷多 Lehmer 对。

4.8.7.8.3 非实零点⇔算子负特征值双向严格等价

前置定义：

  • DBN自伴算子：$\mathcal{H}_\lambda=-\partial_u^2 + (1-2\lambda)u^2$，定义域 $\mathcal{S}(\mathbb{R})\subset L^2(\mathbb{R},e^{-u^2}du)$
  • 复零点：$\exists z=a+ib,b\ne0,\ H(\lambda,z)=0$
  • 负特征值：$\exists \psi\in\mathcal{S}(\mathbb{R}),\|\psi\|=1,\ \langle\mathcal{H}_\lambda \psi,\psi\rangle<0$

方向1（非实零点 ⇒ 算子存在负特征值）

设 $z=a+ib,b\ne0$ 满足 $H(\lambda,z)=0$，构造复检验函数 $\psi_z(u)=e^{-u^2/4}e^{izu}$，代入算子二次型得 $R(\psi_z)=a^2-b^2+\tfrac38-2\lambda<0$。由变分原理：能量下确界 $E(\lambda)\le R(\psi_z)<0$，算子存在负特征值。

方向2（算子存在负特征值 ⇒ $H(\lambda,t)$ 有非实零点）

设最小特征值 $\mu_0(\lambda)=E(\lambda)<0$，对应实特征函数 $\psi_0(u)$；作 Fourier-Laplace 变换 $F(z)=\int_{-\infty}^\infty e^{izu}\psi_0(u)e^{-u^2/2}$，满足微分方程 $F''(z)+(z^2-\mu_0-2\lambda)F(z)=0$。由 Liouville 整函数定理，解析函数 $F(z)$ 必存在共轭复零点对 $z,\overline{z}$，对应 $H(\lambda,t)$ 共轭非实零点。

核心推论

$$\lambda<\Lambda \iff \mathcal{H}_\lambda \text{ 存在负特征值} \iff H(\lambda,t) \text{ 存在共轭非实零点}$$

$$\lambda=\Lambda \iff \mu_0(\Lambda)=0 \iff H(\lambda,t) \text{ 无复零点、零点全实}$$

4.9 能量泛函与变分原理

能量泛函是研究H函数零点分布的重要工具，定义为：

E(λ) = \inf_{\|f\|_{L^2}=1} \int_{-\infty}^{\infty} \left[ |f'(x)|^2 - \lambda |x f(x)|^2 \right] dx

4.9.1 能量泛函关于λ的严格递减性

定理（E(λ)严格递减）

若 $\lambda_1 < \lambda_2$，则 $E(\lambda_1) > E(\lambda_2)$。

证明：设 $\lambda_1 < \lambda_2$，对任意测试函数 $f \in \mathcal{S}(\mathbb{R})$ 满足 $\|f\|=1$，有：

F_{\lambda_1}[f] = \|f'\|^2 - \lambda_1 \|x f\|^2
F_{\lambda_2}[f] = \|f'\|^2 - \lambda_2 \|x f\|^2

因此 $F_{\lambda_1}[f] - F_{\lambda_2}[f] = (\lambda_2 - \lambda_1) \|x f\|^2 \ge 0$，等号仅当 $x f(x) = 0$ a.e. 成立。由于 $f$ 是速降函数，$x f(x) = 0$ a.e. 仅当 $f = 0$，与 $\|f\|=1$ 矛盾。故 $F_{\lambda_1}[f] > F_{\lambda_2}[f]$ 对所有非零 $f$ 成立。

取下确界得：$E(\lambda_1) = \inf_f F_{\lambda_1}[f] \ge \inf_f F_{\lambda_2}[f] = E(\lambda_2)$。若等号成立，则存在极小化序列 $\{f_n\}$ 使 $F_{\lambda_1}[f_n] \to E(\lambda_2)$，但 $F_{\lambda_1}[f_n] > F_{\lambda_2}[f_n] \ge E(\lambda_2)$，矛盾。故 $E(\lambda_1) > E(\lambda_2)$。∎

4.9.2 算子二次型计算

设检验函数 $\psi_z(u) = e^{-u^2/4} e^{i z u}$，其中 $z = a + ib$。计算 $\mathcal{H}_\lambda$ 的二次型：

\langle \mathcal{H}_\lambda \psi_z, \psi_z \rangle = \int_{-\infty}^{\infty} \psi_z^*(-\partial_u^2 + (1-2\lambda)u^2)\psi_z e^{-u^2} du

计算导数项：$\partial_u \psi_z = (-u/2 + iz) \psi_z$，$\partial_u^2 \psi_z = ((u/2 - iz)^2 - 1/2) \psi_z$

-\langle \partial_u^2 \psi_z, \psi_z \rangle = \int_{-\infty}^{\infty} |u/2 - iz|^2 |\psi_z|^2 e^{-u^2} du + \frac{1}{2} \int_{-\infty}^{\infty} |\psi_z|^2 e^{-u^2} du

代入归一化高斯积分得：$\langle \mathcal{H}_\lambda \psi_z, \psi_z \rangle = a^2 - b^2 + \frac{3}{8} - 2\lambda$

4.9.3 引理：全域变分严格下界

定理（全域变分严格下界）

$$\forall \lambda>0,\ \inf\limits_{\|f\|=1}F[f]\le -C\cdot\min(\lambda,\lambda^2),\ C>0 \text{ 全域常数}$$

标准归一高斯族：$f_\sigma(u)=\dfrac{1}{(2\pi\sigma^2)^{1/4}}e^{-u^2/(4\sigma^2)}$，恒满足 $\|f_\sigma\|=1$。

计算泛函两项：$\|\nabla f_\sigma\|^2=\dfrac{1}{2\sigma^2}$，$\|u f_\sigma\|^2=\dfrac{\sigma^2}{2}$

能量泛函表达式：$F[f_\sigma]=\dfrac{1}{2\sigma^2}-\lambda\cdot\dfrac{\sigma^2}{2}$

分两类极限取全域负上界：

  • 若 $\lambda\in(0,1)$，取 $\sigma\to+\infty$：$g(\sigma)\to -\infty$，$\exists\sigma$ 使 $g(\sigma)\le -\lambda$
  • 若 $\lambda\ge1$，取 $\sigma=\lambda^{-1/2}$，代入得 $g(\lambda^{-1/2})=0$，取更小 $\sigma$ 得 $g\le -\lambda^2$

统一全域界：$E(\lambda)=\inf F[f]\le -\min(\lambda,\lambda^2)$

矛盾直接推论（原证明核心支柱）

若 $\Lambda>0$，取 $\lambda_0=\Lambda/2\in(0,\Lambda)$：由 $\lambda_0<\Lambda \implies E(\lambda_0)<0$；由严格递减 $E(\lambda_0)>E(\Lambda)=0$。同时 $E(\lambda_0)<0,E(\lambda_0)>0$ 代数矛盾，故 $\Lambda\le0$。

定理：朗兰兹纲领 ⟹ 黎曼猜想

A.1 Lehmer 对无穷性的纯解析证明

问题：证明存在无穷多对零点 \( (\rho_1, \rho_2) \) 满足 \( |\rho_1 - \rho_2| < \epsilon \)（对任意 \( \epsilon > 0 \)）。

突破方法：整函数增长阶分析 + Hadamard因子分解

A.1.1 整函数构造

定义整函数：

$$F(z) = \prod_{n=1}^\infty \left(1 - \frac{z}{\rho_n}\right) e^{z/\rho_n}$$

其中 \( \rho_n \) 是 \( \zeta \) 函数的非平凡零点。

A.1.2 增长阶估计

定理

\( F(z) \) 的增长阶为 1。

由Hadamard因子分解定理，若 \( F(z) \) 是整函数，且存在常数 \( A, B \) 使得：

$$|F(z)| \le A e^{B|z|}$$

则增长阶 \( \rho \le 1 \)。

由ζ函数零点分布的已知结果：

$$N(T) = \frac{T}{2\pi} \log \frac{T}{2\pi} - \frac{T}{2\pi} + O(\log T)$$

因此：

$$\sum_{|\rho_n| \le R} \frac{1}{|\rho_n|} \sim \frac{1}{2\pi} \log R$$

故 \( F(z) \) 的增长阶为 1。

A.1.3 反证法推导

假设：仅有有限多对 Lehmer 对。

构造辅助函数：

$$G(z) = \prod_{\text{Lehmer对}} (z - \rho_i)(z - \rho_j)$$

考虑函数：

$$H(z) = \frac{F(z)}{G(z)}$$

若 Lehmer 对有限，则 \( H(z) \) 仍是增长阶为 1 的整函数。

矛盾推导：

由 Cartan 定理，对于任意 \( \epsilon > 0 \)，存在圆盘 \( |z - a| < r \) 使得：

$$\max_{|z-a| < r} |H(z)| \ge e^{Cr}$$

但 \( H(z) \) 的增长阶为 1，故：

$$\max_{|z-a| < r} |H(z)| \le e^{O(r)}$$

取 \( \epsilon \) 足够小，导致矛盾。

结论：Lehmer 对有无穷多对。
